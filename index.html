Guia rápido para manutenção

  • Para alterar cores ou fontes edite o arquivo style.css.
  • Os dados e filtros estão no arquivo script.js, dentro da função obterDados().
  • Deseja incluir novas colunas? Adicione o cabeçalho no HTML e depois atualize a função montarLinha() no JavaScript.
  • As seções são independentes, então é seguro remover um bloco inteiro sem quebrar o restante.
