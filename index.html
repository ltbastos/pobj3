Guia rápido para manutenção futura

  • Para alterar cores ou tamanhos, ajuste o bloco de variáveis no início do arquivo style.css.
  • Os dados de exemplo e toda a lógica dos filtros estão em script.js, dentro das funções prepararRegistrosBase e aplicarFiltros.
  • Se precisar acrescentar um filtro, crie o campo aqui no formulário e depois use o mesmo ID para buscar o valor dentro do JavaScript.
  • A tabela aceita qualquer quantidade de colunas. Basta repetir a estrutura <th> no cabeçalho e incluir a célula correspondente em montarLinha.
